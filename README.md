# NNPDE-CPP

For the 1D Burgers equation problem, examples based on the following high-precision numerical method are provided:

- DG-RK3
- FD-RK3-WENO5
- FV-Euler-Godunov
- FV-RK3-WENO5


C++23 is required:

- `std::format` (optional)
- `deducing this` (required for some examples, skip if unsupported)
